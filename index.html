FAQ

Go to Settings › General › VPN & Device Management, find the TunnelSight root CA profile, and tap “Remove Profile.”

Make sure you’ve:
• Installed the Root CA
• Enabled the TunnelSight VPN
• Left the app running — sometimes network handles don’t close until the originating app quits.

We recently updated our JSON schema. Please check our docs for the new field names and structure, or re‑export using the “Legacy JSON” toggle in settings.

Some sites block custom certificates (e.g. fast.com), or enforce HTTP/2‑only. In those cases, TunnelSight can’t decrypt — try disabling HTTP/2 in advanced settings.
